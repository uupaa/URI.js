# Install, Setup modules

```sh
$ brew install closure-compiler

$ git clone git@github.com:uupaa/URI.js.git
$ cd URI.js
$ npm install
```

# Test

```sh
$ npm test
```

# Minify

```sh
$ npm start

  or

$ node node_modules/uupaa.minify.js --keep --output ./lib/URI.min.js ./lib/URI.js
```
